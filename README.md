# Mandible Chemical

A single-page industrial chemical supplier website built with React, Vite, and TailwindCSS.

## Overview

Modern brutalist-styled landing page for Mandible Chemical - a Texas-based supplier of industrial formic acid and sodium bicarbonate for oilfield services, agriculture, apiculture, leather tanning, and toll blending operations.

## Features

- **Bold Brutalist Design**: High-contrast industrial aesthetic with sharp borders and geometric elements
- **Responsive Layout**: Mobile-first design that works across all devices
- **Custom SVG Logo**: Chemical-inspired ant mandible icon
- **Product Showcase**: Detailed specs for Formic Acid and Sodium Bicarbonate
- **Industry Applications**: Six target market segments with specific use cases
- **Contact Form**: Quote request system with industry-specific options

## Tech Stack

- **React 18** - UI framework
- **Vite 5** - Build tool and dev server
- **TailwindCSS 4** - Utility-first styling
- **Lucide React** - Icon system
- **PostCSS** - CSS processing

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open browser to displayed localhost URL (typically http://localhost:5173)

## Build for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

## Deployment

Deploy the `dist/` folder to any static host:
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

## Design Notes

The site uses a **brutalist/neo-brutalist** design system with:
- **Colors**: Zinc grays, orange accents, high-contrast blacks
- **Typography**: Black font weights, uppercase tracking, tighter leading
- **Layout**: Hard borders, geometric shadows, bold sectioning
- **No rounded corners**: Sharp, industrial aesthetic throughouts
